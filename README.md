Estrutura de indexação das regiões, estados e municípios do Brasil.

Objetivo do projeto:
  Listar todos os municípios brasileiros
  Indexar sites de prefeituras
  Mapear portais da transparência
  Permitir monitoramento e cruzamento de dados públicos

Estrutura:
Região → Estado → Municípios

Os dados seguem a divisão territorial utilizada pelo IBGE.
